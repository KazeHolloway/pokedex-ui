# POKéDEX UI

Ce projet consiste en la réalisation d'une interface utilisateur (UI) de POKéDEX développée avec HTML5 et CSS3.

Il a été réalisé dans le cadre des livrables de la semaine 3 de la formation Développeur Fullstack à Akieni Academy.

## Fonctionnalités

- Hero Section (page d'accueil)
- Carte de présentation d'un Pokémon
- Structure HTML5 sémantique
- Mise en forme avec CSS3
- Application du modèle de boîte (Box Model)

## Technologies utilisées

- HTML5
- CSS3
